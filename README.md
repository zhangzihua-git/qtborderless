# qt无边框窗口

## 介绍

- 支持qt无边框窗口
- 支持qt5 qt6
- 依赖c++11

## 功能点

- 无边框
- 有阴影
- 有标题栏
- 标题栏有标准按钮
- 标题栏可移动
- 边框可缩放
- 桌面边缘吸附
- 支持圆角
- 各项功能可选可配置
    - 边框阴影
    - 圆角大小
    - 是否启用边缘吸附
    - 是否启用重置大小
   

## 业务部分
- 业务部分需要放在form中
